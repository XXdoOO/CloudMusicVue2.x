# 组件

## 轮播器

- [ ] 轮播内容可自定义（插槽）
- [ ] 内容数量可自定义
- [ ] 滑动可调节轮播内容
- [ ] 容器中有小圆点并显示当前内容为第几部分
- [ ] 点击小圆点可轮播至该内容
- [ ] 左右有箭头调节
- [ ] 箭头之间的距离由内容宽度决定
